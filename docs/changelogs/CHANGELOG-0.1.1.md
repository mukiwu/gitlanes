# GitLanes 0.1.1

## What's new

- 🎨 New app icon — branching cyan circuit on slate, matching the in-app Vitesse Dark Soft palette.
- 🔧 Fixed updater: `updater:default` and `process:default` capabilities are now granted so the in-app "Check for updates" and 5-second background check actually work.

## Notes

- 第一次發版 (v0.1.0) bundle 還沒接上 icon 設定，所以 macOS 預設灰色佔位 icon。這版補上。
- v0.1.0 沒有給 updater 權限，所以 0.1.0 → 0.1.1 沒辦法自動升級。**從 0.1.0 升的人請手動下載這個 .dmg**；之後從 0.1.1 開始就會正常自動更新。
